1. Spearphishing emails containing weaponized Word documents are sent to targets. 2. The documents contain macros that if enabled install a Delphi executable on the system. 3. The executable collects system info and sends it to a C2 server. 4. The C2 responds with a PowerShell script, likely the POWERSTATS backdoor. 5. POWERSTATS can receive commands from the attackers to exfiltrate files, execute scripts, delete files etc.
