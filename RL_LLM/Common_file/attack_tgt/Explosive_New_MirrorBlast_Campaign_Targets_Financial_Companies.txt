1. The attacker sends a phishing email with a malicious link to the victim. 2. The link delivers a weaponized Excel document with a macro. 3. The macro executes and uses Windows Installer to download an MSI package. 4. The MSI package deploys a Rebol or KiXtart script. 5. The script exfiltrates system data and downloads next stage payloads. 6. Additional malware like FlawedGrace RAT is loaded on the victim's machine. 7. The malware establishes command and control communication for further exploitation.
