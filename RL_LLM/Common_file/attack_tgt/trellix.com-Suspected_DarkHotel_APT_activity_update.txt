 1. The attack starts with a spear phishing email containing a malicious Excel file sent to hotel management. 2. When the Excel file is opened, embedded macros are triggered that execute scripts and scheduled tasks. 3. A VBS script is dropped and executed using wscript.exe to run PowerShell commands. 4. PowerShell is used to collect system info and exfiltrate it to the command and control server fsm-gov.com. 5. A second Excel variant directly executes PowerShell code signed as trusted with SyncappvpublishingServer.vbs. 6. The PowerShell scripts connect to fsm-gov.com to send reconnaissance data from victims.
