1. A phishing email containing weaponized document is distributed. 2. The document contains macros that save it as HTML to access embedded images. 3. A PNG image is converted to BMP to decompress an embedded HTA file. 4. The HTA file is executed using mshta.exe to drop malware. 5. The malware decrypts and loads a RAT payload into memory. 6. The RAT initializes APIs, contacts C2 servers, and can receive encrypted commands. 7. Commands allow process execution, script execution, and data exfiltration.
