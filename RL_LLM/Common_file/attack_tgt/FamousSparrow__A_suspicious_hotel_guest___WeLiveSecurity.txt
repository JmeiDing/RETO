1. FamousSparrow exploits vulnerabilities in public-facing applications like Exchange, SharePoint, and Oracle Opera to gain initial access. 2. The group deploys custom tools including a Mimikatz variant to gather credentials and the SparrowDoor backdoor. 3. SparrowDoor uses DLL hijacking and encrypted shellcode to load itself. 4. The backdoor connects to a C2 server and can receive commands to remotely control the system. 5. SparrowDoor can escalate privileges, dump credentials, discover system info, probe files, exfiltrate data, and more based on C2 commands. 6. Network traffic between SparrowDoor and the C2 is encrypted using symmetric cryptography. 7. FamousSparrow leveraged the Exchange ProxyLogon vulnerabilities to compromise additional targets starting in March 2021.
