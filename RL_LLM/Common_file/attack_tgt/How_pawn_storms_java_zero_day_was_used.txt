1. The attacker sends a spearphishing email containing a malicious URL to the victim. 2. The victim clicks the URL which hosts an exploit that downloads malware. 3. The initial malware is a PE file that drops and executes a DLL component. 4. The DLL collects system information and sends it to a command and control server. 5. In the second phase, the malware downloads additional components like a keylogger. 6. The keylogger sends stolen data back to the command and control server.
