 1. The attackers send a spear-phishing email containing the initial gpUpdates.exe downloader. 2. When executed, gpUpdates drops and executes Distribute.exe. 3. Distribute.exe installs the main svchost.exe RAT and supporting modules like Godown.dll. 4. Svchost.exe starts sniffing packets on port 1234 for commands from the C2 server. 5. The C2 sends commands in UDP payloads to port 1234 which svchost.exe processes. 6. Svchost.exe exfiltrates data like keylogs and screenshots by sending emails via the Naver platform. 7. The operator retrieves data and sends new commands by attaching files to emails.
