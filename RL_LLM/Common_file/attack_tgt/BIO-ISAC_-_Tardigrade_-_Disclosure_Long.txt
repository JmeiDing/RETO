1. The attackers send phishing emails to gain initial access to target networks. 2. They use the Tardigrade malware variant of SmokeLoader as the initial foothold. 3. Tardigrade establishes persistence on systems through boot and login scripts. 4. The malware spreads laterally through network shares and removable media. 5. Tardigrade escalates privileges by impersonating clients and abusing mechanisms. 6. It evades detection using obfuscation, encoding, process injection and other techniques. 7. The malware profiles the network and systems to identify critical assets. 8. It manipulates files and data, and sends stolen information to command servers. 9. Tardigrade can download and deploy ransomware payloads onto compromised networks. 10. The attackers maintain long-term persistence and access to intellectual property.
