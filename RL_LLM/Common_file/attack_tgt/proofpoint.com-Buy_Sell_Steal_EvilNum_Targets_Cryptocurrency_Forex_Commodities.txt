1. The threat actor sends spear phishing emails containing malware to targets. 2. The emails contain ISO, Word, or shortcut file attachments that distribute the EvilNum malware. 3. If victims open attachments, a multi-stage infection chain installs EvilNum. 4. EvilNum fingerprints the antivirus software on the system and selects evasive tactics. 5. The malware executes information stealing and reconnaissance activities on the infected host. 6. EvilNum exfiltrates data from the target network and can be used to deploy additional malware.
