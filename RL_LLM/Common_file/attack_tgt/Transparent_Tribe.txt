1. The attacker sends a spear phishing email containing a document with a link to download a malicious APK file. 2. The user clicks the link which downloads an Android application dropper. 3. The dropper disguises itself as a legitimate app like a porn video player. 4. It checks for targeted apps on the device like WhatsApp and selects one to trojanize. 5. The malicious APK is installed on the device disguising as the legitimate app. 6. The implanted app retrieves the C2 server from a configuration file hosted by the attacker. 7. The malware can now steal contacts, messages, photos, audio, and documents from the infected device.
