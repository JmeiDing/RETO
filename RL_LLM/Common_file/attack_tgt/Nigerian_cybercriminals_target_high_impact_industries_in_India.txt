1. The attackers send spear phishing emails with malware attachments like Pony and Hawkeye. 2. If victims enable macros or execute attachments, the malware infects the system. 3. The malware steals credentials and sends them back to the attacker's command and control server. 4. The attackers manually review compromised accounts to find additional targets. 5. The process repeats using details from compromised accounts to send more convincing spear phishes. 6. The Pony malware also downloads additional modules from the C2 server to further access the infected systems.
