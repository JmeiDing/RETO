1. The attackers send emails with malicious attachments to employees at targeted companies. 2. The attachments are self-extracting archives containing the PoisonIvy malware. 3. When recipients open the attachments, PoisonIvy is installed on their computer. 4. PoisonIvy contacts a command and control server over port 80 using encrypted communication. 5. It provides information to the server like IP address, names of other systems, and password hashes. 6. The attackers use this initial access to move through the network and find systems with valuable data. 7. They copy the targeted data to internal staging servers. 8. Finally, the data is exfiltrated out of the compromised network.
