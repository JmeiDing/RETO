1. The attacker sends a Word document to the victim with a Remote Template reference. 2. When opened, the document retrieves an exploit template from the attacker's server. 3. The exploit template contains code to trigger the CVE-2017-11882 and CVE-2012-1856 vulnerabilities. 4. The exploits deploy the POWERSHOWER PowerShell backdoor malware. 5. POWERSHOWER gathers system info and sends it to the attacker's command and control server. 6. The attacker can send commands to POWERSHOWER to further interact with the infected system. 7. POWERSHOWER cleans up attack artifacts and can download next stage payloads.
