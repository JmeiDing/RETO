1. The malware is executed as a signed binary disguised as a Coinbase job vacancy PDF. 2. It drops files including a PDF, bundle app, downloader agent, and empty file in the Fonts folder. 3. The PDF opens as a decoy while the malware wipes terminal history and extracts binaries from the dropped files. 4. A launch agent is created to persist on the system and execute the downloader. 5. The downloader gathers system info and communicates with a C2 server. 6. The C2 server is contacted to download additional payloads and write them to the empty dropped file.
