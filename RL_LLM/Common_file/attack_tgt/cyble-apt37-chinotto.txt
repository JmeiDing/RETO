 1. The attackers distribute the malware by posing it as a secure messaging application called SecureTalk. 2. When launched, Chinotto requests intrusive permissions like access to SMS, contacts, call logs, accounts, etc. 3. In the background, Chinotto collects sensitive data from the infected device including: - SMS messages - Call logs - Contacts - Device information - External storage files - Account details 4. The malware writes the stolen data to a local text file on the device. 5. Chinotto sends the text file with exfiltrated data to the attacker's command and control server. 6. The malware can also receive commands from the C&C server, allowing the attackers to control infected devices remotely. 7. Chinotto is capable of recording audio and accessing media files on external storage based on commands received from the C&C server.
