 1. Spear phishing emails are sent with malicious attachments, often 7z files containing malware executables. 2. If opened, the malware self-deploys and establishes persistence on the system. 3. The malware collects data including keystrokes, screenshots, credentials, and other sensitive information. 4. Data is exfiltrated via HTTP POSTs to a command and control server IP address. 5. Additional malware may be downloaded, and lateral movement to other systems occurs via tools like PsExec. 6. Compromised accounts are leveraged to access and steal intellectual property and financial information.
