1. The attackers send a spear phishing email with a malicious Microsoft Word document attached. 2. The document exploits CVE-2012-0158 to drop and execute the Kraken malware. 3. Kraken checks if it is running in a VM/sandbox. If not, it sets registry run keys for persistence. 4. It collects system information for reconnaissance and exfiltrates it to a command and control server. 5. Kraken can receive commands from the control panel to further interact with the infected system. 6. It has download and execute capabilities to deploy additional malware stages if the victim is a high-value target.
