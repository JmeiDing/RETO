 1. The attacker sends a spear phishing email with a malicious attachment, often a document with macros or a ZIP file. 2. The attachment executes a dropper module that installs the Crimson RAT malware. 3. Crimson RAT connects to a command and control server operated by the attacker. 4. The attacker sends commands to perform reconnaissance and exfiltrate data from the infected system. 5. Crimson RAT can gather system information, take screenshots, search for files, and steal data. 6. The exfiltrated data is sent to the attacker's server. 7. Crimson RAT provides ongoing backdoor access to the compromised machine.
