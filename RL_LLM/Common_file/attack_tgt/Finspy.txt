1. Attackers distribute FinSpy by sending phishing emails with malicious attachments purporting to be software updates. 2. If targets open attachments, FinSpy installs through complex infection chains tailored for each platform. 3. FinSpy malware connects to command and control servers to receive configurations and modules. 4. Various modules enabled in configuration profile perform surveillance by collecting data like messages, files, microphone recordings, camera images. 5. Collected exfiltrated data is sent back to attacker's command server from the infected device. 6. Attackers monitor and analyze the exfiltrated data to gain insights on the target.
