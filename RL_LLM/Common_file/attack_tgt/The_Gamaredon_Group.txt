1. The attackers send spear phishing emails containing weaponized Office documents as attachments. 2. The victim opens the document, executing embedded macros that launch PowerShell commands. This provides command line access to the system. 3. The attackers use the access to deploy additional malware tools to other endpoints on the network. 4. The attackers study the network, connecting to systems and locating critical servers. 5. The attackers deploy the main remote access trojan (RAT) payload downloaded from their command and control server. 6. The RAT enables the attackers to remotely manipulate the infected hosts.
