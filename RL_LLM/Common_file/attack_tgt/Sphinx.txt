1. The attackers compromise social media accounts related to the targets. 2. They post malicious links in comments on these compromised accounts. 3. The links lead to RAR files containing weaponized Office document files. 4. When victims click on the documents, malware droppers extract modules. 5. The core module starts and reads the configuration file. 6. It injects other modules into various processes based on the config. 7. The modules collect data like keystrokes, screenshots, camera feeds, etc. 8. A communication module sends the stolen data to C2 servers. 9. Lure documents are opened to distract the user from the malware activities. 10. The malware persists via registry keys to survive reboots. 11. The cycle continues as more data is collected and exfiltrated over time.
