1. Gallmaker delivers a malicious Office document to victims via spearphishing email. 2. The document exploits the DDE protocol in Office to execute commands remotely on the victim's system when the victim enables macros. 3. Gallmaker runs various publicly available tools like WindowsRoamingToolsTask, Metasploit payloads, WinZip, and Rex Powershell library on the compromised system. 4. These tools are used to schedule tasks, set up reverse shells, archive data, and manipulate Powershell scripts. 5. Gallmaker communicates with command and control infrastructure to control the compromised hosts. 6. After completing their objectives, Gallmaker deletes tools from the systems to hide traces of their activity.
