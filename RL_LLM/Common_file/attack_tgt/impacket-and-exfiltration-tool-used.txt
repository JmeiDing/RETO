1. APT actors gain initial access to the victim's Microsoft Exchange server in January 2021 using unknown means. 2. The actors use compromised admin credentials to access Exchange and gather info over January-February 2021. 3. In March 2021, the actors exploit Exchange vulnerabilities to deploy webshells. 4. In April 2021, the actors use the Impacket toolkit for lateral movement and access. 5. From July to October 2021, the actors use the custom CovalentStealer tool to exfiltrate sensitive files from the network. 6. The actors maintain persistence likely through stolen credentials until at least January 2022. 7. Throughout the incident, the actors use various tactics like Windows commands, proxy services, and staging to collect data.
