1. Initial Access - The malware disguises itself as the book "The China Freedom Trap" targeting the Uyghur community. - It is distributed outside official app stores to trick users into installing it. 2. Execution - On first launch, the malware checks Android version and hides app icon or shows PDF pages from the book. - It loads all its malicious components and capabilities. 3. Command and Control - The malware connects to the hardcoded command and control server to receive commands. 4. Data Exfiltration - It exfiltrates data like SMS, contacts, call logs, device details, neighboring cell info etc. to the C2 server. 5. Additional Malicious Actions - It can delete logs, take screenshots, access camera to take pictures based on commands. 6. Persistence - The malware persists on the device to continue accepting commands and exfiltrating data.
