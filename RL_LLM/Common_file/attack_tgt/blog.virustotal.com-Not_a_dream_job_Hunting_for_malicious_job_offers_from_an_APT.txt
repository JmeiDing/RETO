1. The threat actor creates weaponized ISO files containing remote access tools like PuTTY. 2. The ISO files have names related to job assessments for companies like Amazon, IBM, Dell. 3. The actor sends emails to victims with lucrative job offers at the companies. 4. The actor follows up over WhatsApp, sharing the malicious ISO as part of the “hiring process”. 5. If the victim mounts the ISO, the remote access tool executes, compromising the system. 6. The tool provides remote access to the threat actor. 7. The actor leverages this access for further objectives like reconnaissance, data theft, lateral movement, etc. 8. Defenders can use VirusTotal to uncover new related ISO samples based on name metadata, volume names, IP addresses, etc. 9. The newly found activity provides insights into the scope of the campaign.
