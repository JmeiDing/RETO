 1. FastPOS is installed via compromised websites, file sharing services, or direct transfer through stolen VNC credentials. 2. The malware copies itself to system folders and adds persistence mechanisms. 3. FastPOS has threads for keylogging, RAM scraping for card data, and self-updating. 4. The keylogger captures keystrokes, while the RAM scraper searches memory and validates card numbers. 5. Valid card data is exfiltrated to a hardcoded C&C server using HTTP GET requests. 6. The C&C receives the stolen card data in plaintext over HTTP. 7. Cybercriminals sell the stolen card data on a website linked to FastPOS C&C IPs.
