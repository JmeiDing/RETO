1. BlackTech group exploits the recently disclosed CVE-2022-1388 vulnerability in F5 BIG-IP devices to compromise Japanese organizations. 2. The attack uses exploit code that enables arbitrary command execution on the BIG-IP devices. 3. The exploit code contains IP addresses of Japanese BIG-IP devices that were targeted. 4. The attackers install malware like TSCookie, Bifrose, and a new Hipid malware variant on the compromised BIG-IP devices. 5. Hipid malware receives commands from C2 servers and can exfiltrate data. Some versions use RC4 encryption. 6. Hipid was also distributed through malicious PyPI packages. 7. The malware disguises its process using techniques like encoding in Base32 and hiding under mount commands.
