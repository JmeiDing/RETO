 1. Ferocious Kitten sends spearphishing emails with weaponized Office documents containing macros. 2. The macros drop and execute the MarkiRAT remote access trojan. 3. MarkiRAT initiates keylogging to capture keystrokes and scrapes memory for data. 4. The malware exfiltrates files of interest back to its C2 server. 5. The C2 server can send commands to control the compromised system. 6. Specific variants target and hijack apps like Telegram and Chrome to gain persistence. 7. The group develops Android malware to target mobile users in Iran as well.
