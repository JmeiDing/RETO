1. Spear phishing emails sent with RTF file attachments containing exploits. 2. Exploits trigger download of malware droppers - files named "8.t". 3. Droppers execute and install next stage malware. 4. Cotx RAT malware decrypts configuration data and beacons to C2 servers. 5. Malware can receive commands from C2 servers to execute on the victim system. 6. Commands include remote shell, file operations, screenshots, etc. 7. Poison Ivy malware follows similar process but uses different droppers and C2 infrastructure. 8. Overlapping targeting and infrastructure links Cotx RAT and Poison Ivy campaigns.
