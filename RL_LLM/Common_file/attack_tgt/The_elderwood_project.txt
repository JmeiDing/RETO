 1. The group identifies zero-day vulnerabilities in common software like Adobe Flash, Internet Explorer, etc. 2. They develop exploits and malware payloads for the vulnerabilities. 3. The malware profiles the victim system and organization for further attacks. 8. Additional malware like keyloggers is downloaded to steal sensitive data from interesting victims. 9. The malware exfiltrates stolen data back to the attacker's infrastructure.
