 1. A decoy executable file drops and loads an encrypted USBferry malware component. 2. The malware injects code into the rundll32.exe process. 3. USBferry loads its configuration and copies itself to any connected USB drives. 4. When the USB drive is connected to an air-gapped system, USBferry infects it and collects data. 5. The data is exfiltrated if the USB drive is connected to an internet-enabled system. 6. The exfiltrated data is sent to the command and control server. 7. USBferry establishes persistence on infected systems.
