 1. Spearphishing - Send emails with malicious Office docs to targets. 2. Macro Execution - Macros in the documents run PowerShell scripts. 3. Malware Delivery - PowerShell downloads payloads from attacker domains. 4. Execution - Payloads execute, installing remote access trojans and keyloggers. 5. Command & Control - Malware connects to C2 servers to receive commands. 6. Data Exfiltration - Keyloggers and RATs send stolen data to attacker servers.
