1. The attackers send spear phishing emails with malicious attachments. 2. The attachments infect machines with Dridex malware or Cobalt Strike beacons. 3. Dridex and Cobalt Strike are used to move laterally and infect more machines on the network. 4. The attackers spend time exfiltrating data from the victim network. 5. The Entropy ransomware DLL is copied to machines across the network using batch scripts. 6. The ransomware is executed on the compromised machines via a DLL hijacking technique. 7. The ransomware encrypts files on the infected machines. 8. A ransom note is displayed demanding payment in exchange for the decryption key.
