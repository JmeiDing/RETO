 1. The malware is distributed through phishing emails with Excel document attachments. 2. The attachment is executed by the victim which installs Aveo and opens the decoy document. 3. Aveo sends system information to its command and control server. 4. The attackers can then send commands to control the compromised system. 5. Aveo persists on the victim's system to maintain access for the attackers. 6. The attackers leverage this access to further compromise assets and steal data.
