1. The malware is delivered via a weaponized Microsoft Office document exploiting a vulnerability like CVE-2012-0158. 2. When the document is opened, it drops a file named "svohost.exe" in the user's temp folder and executes it. 3. "svohost.exe" is the Troj/ReRol.A malware. It copies itself to the user's Application Data folder. 4. The malware contacts the command and control server at mac.avstore.com.tw. 5. It collects system information like installed software, services, IP configuration etc. and sends it to the C2. 6. The C2 responds by sending additional malware to execute on the infected machine. 7. The malware achieves persistence by adding a registry run key to execute on reboot.
