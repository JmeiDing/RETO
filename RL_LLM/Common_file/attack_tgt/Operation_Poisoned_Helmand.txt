 1. The attackers compromise a JavaScript file hosted on a content delivery network (CDN) used by Afghan government websites. 2. When visitors load any of the government websites that use the CDN, the malicious script is loaded. 3. The script redirects the victim's browser to download a Java applet from a specified URL. 4. The Java applet contains malicious code that downloads and decodes an executable payload. 5. The executable payload drops multiple files including a malicious DLL and encrypted backdoor. 6. The backdoor connects to a command and control server to allow the attackers to control the infected system.
