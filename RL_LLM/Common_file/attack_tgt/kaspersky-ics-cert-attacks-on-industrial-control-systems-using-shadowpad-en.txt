1. The attackers exploited CVE-2021-26855 in Microsoft Exchange to gain initial access. 2. The ShadowPad backdoor was installed disguised as mscoree.dll and launched using AppLaunch.exe. 3. ShadowPad was later installed using DLL side-loading in the OleView utility. 4. The attackers ran manual commands for reconnaissance and data theft. 5. Automated data collection was set up using scripts that collected files, credentials, USB history, etc. 6. Additional tools like CobaltStrike, Mimikatz, PlugX, and webshells were deployed. 7. Data was exfiltrated over the command and control channels. 8. Infrastructure analysis revealed domains and servers tied to the campaign.
