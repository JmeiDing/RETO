 1. The attackers send phishing emails with malicious links or attachments to targets. 2. If the victim clicks the link or opens the attachment, the first stage malware dropper is downloaded. 3. The obfuscated dropper executes and decrypts the RAT payload. 4. The dropper injects the RAT into a legitimate system process for execution. 5. The RAT connects to a C2 server to receive commands. 6. The RAT can perform keylogging, screen capturing, file exfiltration, and other spying activities on the infected machine. 7. The RAT sends any collected data back to the C2 server for the attackers.
