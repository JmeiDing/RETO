 1. Reconnaissance - Attackers scan and gather intelligence on the target using open sources. 2. Breach - Attackers exploit vulnerabilities to gain initial access to the network. 3. Foothold - Malware like afset and setMFT are used to hide the attacker's presence. 4. Move Laterally - Attackers use stolen admin credentials and remote access tools to expand access. 5. Exfiltrate - Data is extracted over time using VPN accounts and remote access tools. 6. Delete Tracks - Logs are erased and malware like Destover deployed to destroy data. 7. Exit - Stolen data is published online and destructive malware is activated as attackers withdraw.
