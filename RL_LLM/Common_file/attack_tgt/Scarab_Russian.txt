1. The attacker compromises a website or sets up a fake domain. 2. The attacker sends spear phishing emails with malicious attachments to selected Russian-speaking targets. 3. The attachment contains an exploit that triggers a vulnerability like CVE-2012-0158 when opened. 4. The exploit executes and installs Trojan.Scieron malware on the victim's machine. 5. Trojan.Scieron collects system information and downloads Trojan.Scieron.B. 6. Trojan.Scieron.B installs itself and uses its rootkit component to hide malicious network activity. 7. The malware collects data and awaits commands from the attacker who now has remote access. 8. The attacker can now move laterally to compromise additional systems on the network.
