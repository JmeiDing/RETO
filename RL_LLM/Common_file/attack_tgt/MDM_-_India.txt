1. Attacker enrolls victim's iPhone into a malicious MDM server through social engineering or physical access. 2. The MDM profile is installed on the device, allowing remote management. 3. Attacker uses MDM to uninstall messaging apps like Telegram and WhatsApp. 4. Attacker distributes compromised versions of the apps with injected malicious code. 5. The trojanized apps steal data like messages, contacts, location, and exfiltrate it to a C2 server. 6. The malware also leverages background execution and scheduling to persist on the device. 7. Stolen data is sent to the techwach[.]com C2 server controlled by the attacker.
