1. Spam emails are sent with malicious Word document attachments containing macros. 2. The victim opens the Word document, which executes the macro. 3. The macro runs a PowerShell command to download and execute the next stage PowerShell script. 4. The PowerShell script downloads and executes the GandCrab ransomware loader. 5. The GandCrab loader injects the GandCrab ransomware executable into memory. 6. The PowerShell script also downloads the Ursnif malware executable. 7. Ursnif executes to steal credentials and distribute other malware.
