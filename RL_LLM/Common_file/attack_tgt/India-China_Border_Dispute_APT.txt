1. The attackers send a spearphishing email with a Word doc about India-China border tensions. 2. The document contains a macro that executes PowerShell code. 3. The PowerShell code runs embedded shellcode in memory. 4. The first stage shellcode downloads a second stage shellcode pretending to be from Microsoft. 5. The second shellcode contains an encrypted Cobalt Strike beacon. 6. The shellcode decrypts and executes the beacon payload. 7. The beacon connects to C2 servers using DGA domains and steganography.
