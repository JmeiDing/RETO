 1. FastViewer is installed on the victim's device, disguised as the legitimate Hancom Office Viewer app. 2. When the victim opens a specially crafted document file, FastViewer checks the first 4 bytes for the string "EDC%". 3. If the string is present, it transmits device information to the C&C server. 4. FastViewer receives a response from the C&C server with instructions to download the FastSpy module. 5. FastViewer downloads and decrypts the FastSpy module and saves it to the device. 6. FastSpy establishes communication with the C&C server and can receive commands to take control of the infected device.
