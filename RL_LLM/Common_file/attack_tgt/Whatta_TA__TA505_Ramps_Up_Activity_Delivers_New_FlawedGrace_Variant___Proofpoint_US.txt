 1. TA505 sends emails with Excel attachments or links to Excel files. 2. The Excel file macros download and execute an MSI file when opened. 3. The MSI file runs a KiXtart script loader. 4. The KiXtart script contacts the C2 and is instructed to download another MSI file. 5. The second MSI file executes a Rebol script loader called MirrorBlast. 6. MirrorBlast downloads additional Rebol scripts. 7. The Rebol scripts execute the ReflectiveGnome loader. 8. ReflectiveGnome downloads shellcode. 9. The shellcode drops and executes the FlawedGrace RAT. 10. FlawedGrace connects to C2 servers to receive commands.
