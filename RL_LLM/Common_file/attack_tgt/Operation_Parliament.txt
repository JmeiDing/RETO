1. The attackers send spearphishing emails with weaponized documents to high-value targets. 2. If the document is opened, malware is installed on the victim's system. 3. The malware establishes an encrypted C&C channel to the attacker's server. 4. The C&C server sends encrypted scripts and commands to the malware. 5. The malware executes the received scripts and commands on the infected system. 6. Results of the scripts/commands are sent back to the C&C server via the malware. 7. The attackers use this to steal data and access systems within the victim organization.
