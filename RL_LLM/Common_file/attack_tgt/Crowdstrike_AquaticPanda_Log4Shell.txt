1. AQUATIC PANDA gained initial access to the vulnerable VMware Horizon server at the academic institution, likely using a modified Log4Shell exploit. 2. The threat actor performed reconnaissance using DNS lookups to identify other vulnerable servers. 3. AQUATIC PANDA attempted to execute Linux commands and download additional tools onto the Windows server. 4. The group retrieved malicious payloads including a reverse shell and a credential dumping tool. 5. AQUATIC PANDA dumped credentials from the LSASS process memory using native OS tools. 6. The dumped credentials were compressed and prepared for exfiltration. 7. The threat actor attempted to delete evidence of their presence on the system. 8. OverWatch detected the suspicious activity early in the intrusion and quickly notified the institution. 9. The institution was able to implement incident response, patching the vulnerability and preventing further access.
