 1. Tick sends spearphishing emails containing weaponized attachments to targets. These attachments exploit vulnerabilities to install malware. 2. The initial malware payload connects to command and control servers hosted on compromised websites. 3. The command and control servers instruct the malware to download additional components like Daserf, Gh0st RAT, etc. 4. The malware establishes persistence on the infected system and exfiltrates data. 5. In some cases, the malware downloads next stage downloaders like HomamDownloader that then retrieve more malware. 6. The additional malware like Daserf and Gh0st RAT enable remote access, command execution, and data exfiltration. 7. Tick tailors social engineering tactics like the email attachment technique to the target's expected behavior.
