1. The malware loader is distributed via pirated software installers from a malware-as-a-service platform. 2. The loader installs the main PseudoManuscrypt module, which adds itself as a service to persist on the system. 3. The module steals data like keystrokes, screenshots, microphone recordings, VPN credentials, etc. and sends it to a command server. 4. The command server can send back commands to perform additional malicious actions like disabling security software.
