1. The attackers gain initial access by brute forcing Remote Desktop Protocol credentials using penetration testing tools. 2. They use additional hacking tools to move laterally and maintain access, like password stealers and remote access trojans. 3. The Gorynych malware is installed to steal credentials and keystrokes, and specifically target Bitcoin wallets. 4. Gorynych downloads and executes the BlackPOS RAM scraper to steal payment card data from POS systems. 5. The modular nature of Gorynych allows adding other capabilities like taking screenshots, keylogging, spreading via USB, etc. 6. Gorynych communicates with command and control servers to receive configurations and exfiltrate stolen data. 7. The attackers plant other POS malware like NewPOSThings, Neutrino, and Project Hook.
