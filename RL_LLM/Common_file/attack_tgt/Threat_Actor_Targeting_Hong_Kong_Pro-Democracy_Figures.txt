 1. The threat actor sends spear phishing emails containing Google Drive links to ZIP archives. 2. The ZIP archives contain benign documents and a weaponized LNK file. 3. The LNK file executes msiexec.exe to download a remote MSI file. 4. The MSI file drops and executes a Python-based malware executable. 5. The malware collects system info and installs an autorun registry key for persistence. 6. It uses the Dropbox API as a C2 channel to receive encrypted commands from the attacker. 7. The malware executes the commands and exfiltrates any results back to the C2 server.
