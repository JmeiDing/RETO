1. Attacker sends emails with HTML attachments containing refresh tags to redirect to URLs. 2. The URLs redirect to cloud storage sites hosting ZIP archives. 3. The ZIP archive contains a PE file with an embedded HTA script. 4. User double clicks the PE file, which extracts and executes the HTA. 5. The HTA fetches a VBS script that downloads the final ZIP payload. 6. The ZIP payload contains a trojanized Windows tool and banking trojan DLL. 7. The VBS renames files to load the trojan DLL via DLL side-loading when the tool runs. 8. The banking trojan steals credentials and data and connects to C2 servers.
