1. A first stage malware like Skipper drops the Gazer payload on victims. 2. The Gazer loader injects the orchestrator module into explorer.exe. 3. The orchestrator decrypts config data and loads other modules from resources. 4. Modules gather recon data, steal browser files, spread via USB, etc. 5. The communication module injects into a browser process for internet access. 6. The C2 server sends encrypted commands which modules execute. 7. Data theft results are encrypted and exfiltrated back to the C2. 8. The C2 can push new modules, configurations, and tasks to infected hosts.
