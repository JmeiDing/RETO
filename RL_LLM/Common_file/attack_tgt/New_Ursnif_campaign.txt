1. The attacker sends a Word document named info_03_24.doc containing VBA macros. 2. The VBA macros execute on open and generate strings, copy mshta.exe, write an HTML file named index.html, and execute it. 3. The index.html file contains obfuscated JavaScript and ActiveX code. 4. The JavaScript decodes a function that contains the URL for the final payload DLL. 5. ActiveX is used to download the DLL payload from the decoded URL. 6. The downloaded DLL is saved as index.dll in the temp folder. 7. The DLL is executed using regsvr32.exe. 8. The DLL contains the Ursnif banking Trojan that executes in DllMain.
