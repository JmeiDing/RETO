PAM functions are hooked to allow remote authentication with a hardcoded password. 8. Encrypted C2 communications allow backup remote access. 9. Symbiote targets financial organizations in Latin America using impersonated bank domains for C2. 10. Samples were submitted to VirusTotal likely for testing detection before deploying malware.
