1. Actors send a phishing email with a malicious Office document containing macros. 2. The macro executes PowerShell to download and run the Powersniff malware. 3. Powersniff performs reconnaissance and contacts a command and control server. 4. The C2 sends commands to deploy the ShellTea backdoor. 5. ShellTea injects into Explorer and uses API hashing and other evasion techniques. 6. The actors move laterally and install ShellTea on staging servers. 7. The PoSlurp POS malware is deployed to target systems. 8. PoSlurp injects into POS processes to scrape payment card data from memory. 9. The scraped data is encrypted and exfiltrated to the attackers. 10. The malware is cleaned up after data exfiltration.
