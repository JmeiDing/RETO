1. The attackers send spearphishing emails with malware to targets. 2. They exploit vulnerabilities in VPN services like Pulse Secure to gain initial access. 3. Privilege escalation tools like JuicyPotato are used to gain admin rights. 4. Credentials are dumped from memory using ProcDump and Mimikatz. 5. Self-developed backdoors open SSH tunnels to enable RDP connections. 6. Webshells are installed on internal servers for command execution. 7. Tools like Ngrok provide external access to internal servers. 8. DNS tunneling and reverse proxies enable C2 communications. 9. Reconnaissance is done to identify valuable data on internal networks. 10. Stolen files are filtered, compressed, and exfiltrated via the C2 channels.
