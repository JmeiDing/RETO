The PDF describes analysis of a malware family called BBSRAT that is linked to previous attacks dubbed 'Roaming Tiger' targeting Russia/Russian-speaking countries. BBSRAT is delivered via weaponized RTF documents exploiting CVE-2012-0158. It uses sideloading to load a malicious DLL (pnipcn.dll) using a legitimate Citrix binary (ssonsvr.exe). The DLL injects into ssvchost, launches powershell to download additional payloads reflectively. It communicates with C2 servers using custom encrypted protocol over HTTP/HTTPS. It supports various commands like file operations, process control, command execution, etc. It exfiltrates system info and is used for espionage. Overlaps found with previous C2 infrastructure used in Roaming Tiger campaigns.
