The report analyzes a new campaign by the threat actor Kimsuky distributing the remote access trojan xRAT. Initial Infection - Kimsuky installs a variant of the Gold Dragon malware that establishes persistence and downloads additional payloads. Malware Capabilities - Gold Dragon variant lacks info-stealing capabilities, so xRAT is installed to enable remote control and data exfiltration. xRAT Analysis - xRAT is an open-source RAT that Kimsuky delivers via process hollowing to avoid detection. It communicates with a C2 server. Cleanup Tool - An additional executable is distributed to remove traces of the attack from infected systems. Detection - IOCs for the installer, malware samples, C2 servers, and cleanup tool are provided to detect this new Kimsuky activity.
