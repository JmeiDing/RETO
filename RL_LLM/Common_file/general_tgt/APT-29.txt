The report describes recent activity of APT29 group targeting COVID-19 vaccine research. APT29 used exploits against Citrix, Pulse Secure etc. to gain initial access to target networks. Custom malware WellMess and WellMail used for C2 communications after initial infection. WellMess is a lightweight malware for command execution, file uploads/downloads. WellMail runs commands and scripts sending results to a hardcoded C2 server. The group uses a variety of techniques including initial infection vectors, spear-phishing, and certificate usage. They have been successful using recently published exploits to gain initial footholds.
